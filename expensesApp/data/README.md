# Data Directory

This directory stores the SQLite database file (`expenses.db`).

⚠️ **Important**: Database files are excluded from git as they contain personal financial data.

## Backup

Make sure to backup this directory regularly! You can:
- Copy the entire `data/` folder to a secure location
- Use cloud storage (encrypted)
- Schedule automated backups

## Database File

- `expenses.db` - Main SQLite database containing all your expense records
